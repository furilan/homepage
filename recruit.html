AIエンジニア

正社員 / 業務委託

LLMを活用したAIエージェントの設計・開発をリードしていただきます。プロンプト設計、RAG構築、外部システム連携など、AI活用の最前線に携われるポジションです。

  • Python
  • LLM / RAG
  • API設計
  • クラウド

ITエンジニア(SES)

正社員

お客様先のプロジェクトに参画し、システム開発を推進していただきます。経験・スキルに応じたプロジェクトへのアサインとキャリア支援を行います。経験の浅い方のご応募もご相談ください。

  • Web開発
  • 業務システム
  • インフラ
  • テスト

Webエンジニア(自社サービス)

正社員 / 業務委託

「フリランドットコム」「なまらワークス」をはじめとする自社サービスの開発・改善を担当していただきます。企画から実装・運用まで、サービスづくり全体に関われます。

  • フロントエンド
  • バックエンド
  • UI/UX
  • グロース

セールス

正社員

AIエージェント開発・SES・自社サービスの営業を担当していただきます。新規開拓から既存顧客のフォローまで、お客様の課題をヒアリングし、最適なソリューションを提案する役割です。IT・AIへの興味があれば、業界未経験の方も歓迎します。

  • 新規開拓
  • 提案営業
  • 顧客折衝
  • IT/AI

コンサルタント

正社員 / 業務委託

お客様のDX推進やAI活用を上流から支援していただきます。課題の整理から要件定義、技術選定、プロジェクト推進まで、ビジネスとテクノロジーの両面から伴走するポジションです。

  • DX推進
  • AI活用支援
  • 要件定義
  • PM/PMO

マーケター

正社員 / 業務委託

自社サービスや会社全体の認知拡大・リード獲得を担っていただきます。Webマーケティング、コンテンツ制作、SNS運用、広告運用など、施策の企画から実行・分析まで幅広く携われるポジションです。AIを活用したマーケティングにも挑戦できます。

  • Webマーケ
  • コンテンツ
  • SNS運用
  • データ分析

企画営業

正社員

お客様の課題やニーズをもとに、新しいサービス・ソリューションを企画し、提案・実現まで一気通貫で担っていただきます。単なる御用聞きではなく、「何をつくり、どう届けるか」を自ら描いていく、事業づくりに近いポジションです。

  • サービス企画
  • 提案営業
  • 事業開発
  • 顧客折衝
